Add Hibernte to shutdown button : 1- On the taskbar, type control panel, and select it. 2- Select System and Security. 3- In the Power Options section, select Change what the power buttons do. 4- Select Change settings that are currently unavailable. 5- In the Shutdown settings section, select Hibernate. 6-Select Save changes. Temporarily swith off windows security : Start > "Windows Security" > under Virus & threat protection settings > select Manage settings > Switch Real-time protection to Off.
