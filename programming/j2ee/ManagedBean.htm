. Managed Bean is a regular Java Bean class registered with(managed by) JSF.
	. From JSF 2.0 onwards, Managed beans can be easily registered using annotations. 
	. The presence of this annotation on a class automatically registers the class with the runtime as a managed bean class.
	. The managed bean contains the getter and setter methods, business logic or even a backing bean
		(a bean contains all the HTML form value).
	. Managed beans works as a Model for UI component, so it can be accessed from JSF page.
	. a Managed bean can have scope :request ,view ,flash,session,aplication,server.
	. You could  use  managed beans as the model or the glue between the controller and the model.
