laravel

installation
	composer create-project laravel/laravel –-prefer-dist
	for complete installation:
	composer create-project laravel/laravel test dev-develop
starting laravel service
	php artisan serve
running:
	http://localhost:8000
Basic config:
	1. set the write permission for the directory "storage" and "bootstrap/cache".
	2. Generate Application key to secure session and other encrypted data.
	 If the root directory doesn’t contain the .env file then rename the .env.example to .env file
	 and execute the following command where you have installed Laravel:
    
    php artisan key:generate 
    
Database Configuration
	 can be configured from config/database.php file:
	  (host ,database,username ,password ,etc )
Naming the Application
	The App Directory, by default, is namespaced under App.
	To rename it, you can execute the following command \
	and rename the namespace:

	php artisan app:name 
	
BasicRouting
	routing is meant to route your request to an appropriate controller. 
	The routes of the application can be defined in app/Http/routes.php file.
	Here is the general route syntax for each of the possible request.

	Route::get('/', function () {
		return 'Hello World';
	});
	Route::post('foo/bar', function () {
		return 'Hello World';
	});
	Route::put('foo/bar', function () {
	//
	});
	Route::delete('foo/bar', function () {
	//
	});
	
	Example
		app/Http/routes.php
