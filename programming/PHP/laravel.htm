Installation composer create-project laravel/laravel –-prefer-dist Complete installation and creating first boilerplate project: 1.create basic project: composer create-project --prefer-dist laravel/laravel my-blog 2.create database: Go to http://localhost/phpmyadmin/ and Create a new Database by clicking on new (shown below in red) Name it my_blog and click Create 3. set configurations: 3.1. Open your file explorer and navigate to my-blog folder 3.2. Open the .env file in your code editor 3.3. Change the following in the file:- APP_NAME key to name of your blog i.e. “My Blog” DB_DATABASE key to database name i.e. my_blog starting laravel service php artisan servecode this will run the project at http://localhost:8000 BasicRouting routing is meant to route your request to an appropriate controller. The routes of the application can be defined in app/Http/routes.php file. Here is the general route syntax for each of the possible request. 

	Route::get('/', function () {
		return 'Hello World';
	});
	Route::post('foo/bar', function () {
		return 'Hello World';
	});
	Route::put('foo/bar', function () {
	//
	});
	Route::delete('foo/bar', function () {
	//
	});
